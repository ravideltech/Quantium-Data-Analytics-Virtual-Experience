# Quantium Customer Analytics (Virtual Experience Program)

## Project Overview

This project was completed as part of the Quantium Data Analytics Virtual Experience Program.

The objective was to analyze customer purchasing behaviour using transaction and customer datasets to identify purchasing patterns, customer segments, and provide commercial recommendations for the Category Manager.

---

## Business Problem

Quantium's Category Manager wanted to understand:

- Which customer segments contribute the highest sales?
- Which chip brands perform best?
- Which pack sizes are most popular?
- How can marketing campaigns be better targeted?

This project answers these questions through exploratory data analysis and customer segmentation.

---

## Dataset

Two datasets were analyzed:

- Transaction Data
- Customer Purchase Behaviour Data

The transaction dataset contains over **264,000 transactions** from **72,000+ loyalty customers**.

## Key Findings

- Older Singles/Couples generated the highest total sales.
- Mainstream customers contributed the largest revenue.
- Kettle was the highest-selling chip brand.
- 175g was the most popular pack size.
- Older Families purchased the highest average quantity.

---

## Business Recommendations

- Focus promotions on Older Singles/Couples and Mainstream customers.
- Maintain inventory for Kettle products.
- Prioritize 175g and 150g pack sizes.
- Introduce bundle offers for popular brands.
- Design customer-specific loyalty promotions.
